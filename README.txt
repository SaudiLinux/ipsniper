# أداة فحص الثغرات الأمنية باستخدام Nessus

تم تطوير هذه الأداة بواسطة: Saudi Linux
البريد الإلكتروني: SaudiCrackers@gmail.com

## المتطلبات الأساسية

1. تثبيت Python 3.x
2. تثبيت Nessus Scanner
   - قم بتحميل Nessus من الموقع الرسمي: https://www.tenable.com/downloads/nessus
   - اتبع خطوات التثبيت
   - قم بتشغيل خادم Nessus

## تثبيت المكتبات المطلوبة

قم بتثبيت المكتبات المطلوبة باستخدام الأمر التالي:
```
pip install requests colorama
```

## إعداد الأداة

1. قم بتشغيل Nessus وافتح المتصفح على العنوان: https://localhost:8834

2. قم بإنشاء حساب وتسجيل الدخول

3. احصل على مفاتيح API:
   - اذهب إلى Settings > My Account
   - اضغط على Generate API Keys
   - احفظ مفتاح الوصول (Access Key) والمفتاح السري (Secret Key)

4. قم بتعديل المتغيرات في ملف nessus_scanner.py:
   - url: عنوان خادم Nessus (الافتراضي: https://localhost:8834)
   - access_key: مفتاح الوصول الخاص بك
   - secret_key: المفتاح السري الخاص بك

## تشغيل الأداة

1. افتح موجه الأوامر (Terminal)

2. انتقل إلى مجلد الأداة:
   ```
   cd path/to/tool/directory
   ```

3. قم بتشغيل الأداة:
   ```
   python nessus_scanner.py
   ```

4. اتبع التعليمات على الشاشة:
   - أدخل الهدف للفحص (عنوان IP أو نطاق)
   - أدخل اسم الفحص

5. ستقوم الأداة بـ:
   - إنشاء فحص جديد
   - بدء عملية الفحص
   - عرض حالة التقدم
   - حفظ النتائج في ملف JSON
